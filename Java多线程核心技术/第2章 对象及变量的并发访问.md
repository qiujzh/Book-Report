### 2.1 synchronized同步方法 ###
“非线程安全问题”出现于多个线程并发访问同一个对象的实例变量时，产生的后果是“脏读”。如果是方法内部的变量，则不存在“非线程安全问题”。   
``` java   
public class HasPrivateNum {
	private int num = 0;
	public void setNum(String userName) {
		if(userName.equals("a")) {
			num = 100;
		}else {
			num = 200;
		}
		System.out.println(userName + "   " + num);
	}
}

```
