## 1.1进程和线程 ##
### 进程 ###

	1. 进程是操作系统分配资源的最小单位
	
	2. 进程是操作系统结构的基础，是一次程序的执行，是一个程序及其数据在处理机上顺序执行时所发生的活动，是程序在一个数据集合上运行的过程
	
	3. 举例：QQ是一个进程
### 线程 ###
	1. 线程是操作系统调度的最小单位
	
	2. 线程是进程中独立运行的子任务
	
	3. 举例：QQ中的聊天、文件下载是线程
